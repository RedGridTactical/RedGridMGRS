## Red Grid Tactical v1.0.0

First public release.

### What's in this release

**GRID tab**
- Real-time MGRS position display to 1-meter precision
- Accuracy and altitude readout
- Wayfinder arrow with bearing (declination-corrected) and distance to waypoint
- Waypoint entry by MGRS coordinate or mark current position
- Full portrait and landscape layout support

**TOOLS tab — 8 tactical calculators**
- **Back Azimuth** — reciprocal bearing with optional declination correction
- **Dead Reckoning** — estimate new position from heading and distance
- **Resection** — determine your position from two known points and compass bearings
- **Pace Count** — calibrate paces/100m, convert paces ↔ distance
- **Declination** — save local magnetic declination, convert magnetic ↔ grid bearings
- **Time / Distance / Speed** — travel time and ETA from terrain speed presets
- **Sun / Moon** — live solar and lunar azimuth and elevation for field orientation
- **MGRS Precision** — view your position at all 5 reporting precision levels

**REPORT tab — 3 radio-ready templates**
- **SALUTE** — enemy contact report, grid auto-filled
- **9-Line MEDEVAC** — medical evacuation request, pickup grid auto-filled
- **SPOT Report** — observation report, location and DTG auto-filled
- One-tap COPY formats the complete report for radio transmission

### Privacy
- Zero network requests
- Zero location data stored
- Only two user settings saved locally: declination offset and pace count
- Location permission: foreground only

### Installation
- **Android:** Download `RedGrid-Tactical.apk` below. Enable "Install from unknown sources" when prompted.
- **iOS:** Download `RedGrid-Tactical.ipa` below. Install via AltStore (altstore.io).

### Permissions
- Location (while using app) — foreground only, for MGRS display
